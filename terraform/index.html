Project Summary

This project demonstrates how modern cloud infrastructure can be deployed through Infrastructure as Code instead of manual configuration. The environment includes AWS networking, compute, security, monitoring, containerization, and Linux server administration.

Infrastructure as Code

Terraform was used to define and deploy AWS infrastructure ensuring consistent, repeatable deployments.

AWS Networking

Configured a VPC, public subnet, internet gateway, route table, and security group.

Compute & Hosting

Deployed the application on an Amazon EC2 Linux server with public web access.

Monitoring

Used Amazon CloudWatch to monitor EC2 infrastructure performance and health.
